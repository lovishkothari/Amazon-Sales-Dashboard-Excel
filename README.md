# Amazon Sales Dashboard and Data Analysis in Excel

## Project Overview
This project is an Excel-based sales analysis and dashboard project created using Amazon sales data. The main objective of this project is to analyze sales performance, customer trends, regional targets, and product insights using Excel analytics and dashboarding techniques.

The project includes data cleaning, pivot table analysis, KPI tracking, interactive dashboard creation, and business insight generation.

---

## Dataset Information
The project uses multiple datasets including:

- Amazon Sales Data
- Product Master Data
- Customer Master Data
- Regional Goals Data

---

## Objectives
- Analyze Amazon sales performance
- Track revenue and sales trends
- Compare regional performance
- Identify top-performing products
- Monitor customer-related insights
- Build an interactive Excel dashboard
